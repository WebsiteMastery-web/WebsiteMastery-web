# Hi, I'm Sam Kleiman 👋

I'm a full-stack developer and entrepreneur based in **Buenos Aires, Argentina**, building two businesses at the intersection of AI, telephony, and sales automation.

---

## What I'm Building

### 🏢 Custom CRM Platform
A multi-tenant CRM system built from the ground up on **Supabase + Next.js**, designed for service businesses and sales agencies. Features include pipeline management, role-based access control, call log integration, and n8n-powered workflow automation.

### 📞 AI Voice Calling Agency
An outbound/inbound AI calling infrastructure using real-time voice models and telephony APIs (Telnyx). Clients get automated calling agents that handle lead qualification, follow-ups, and appointment booking — synced directly into the CRM.

Together, these two products form a fully integrated sales stack: the CRM manages the data and the voice agents work the pipeline automatically.
