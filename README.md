# Sam Kleiman

Building Limitless AI — an autonomous multi-agent financial intelligence system powered by Claude.

## What I'm working on

**Limitless AI**: A 12+-layer multi-agent system that synthesizes live market data, on-chain flows, geopolitical signals, crowd psychology, and more to make high-conviction trading decisions.

It runs parallel intelligence layers, adversarial debates (Bull vs Bear vs Neutral), and a final Portfolio Manager (Claude Opus) before executing any trade. Designed to operate at a cognitive level no single human analyst can match.

Built for the exact finance use cases Anthropic is most excited about — where Claude fills genuine gaps that humans can't handle alone.

## Current Build (Limitless AI)

- 12+ parallel intelligence layers (market data, whale tracking, GDELT geopolitics, crowd simulation, regime detection, etc.)
- Multi-agent debate engine with Bull, Bear, and Neutral analysts + neutral arbiter
- Claude Opus Portfolio Manager with confidence-gated decision making
- Risk management, Elo-weighted agent ranking, and self-blocking safety mechanisms
- Live paper trading + backtesting pipeline

**Status**: Actively running paper trades on BTC and Polymarket. Backtested with real historical verification.
