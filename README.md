## What I'm working on

A CRM platform and AI calling infrastructure that work together as a single system.
The CRM handles pipeline, contacts, and data. The voice layer works the pipeline
automatically — qualifying leads, following up, booking appointments — without
human intervention.

Both products are built to replace the stack most agencies pay $2,000+/month
to stitch together manually.

- Multi-tenant CRM — RLS, pipeline views, call tracking, workflow automation
